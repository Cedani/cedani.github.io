TestPr

Project Description

TestPr is a project that I started in my second year at Epitech Montpellier. The goal of the program is to easy and speed the process of making unit test for project in C with criterion unit test library.
The languages used are C++ and QT

Explanation

In this video, I selected a project via my program, and i have generated a new test in the file test1.c I have also generated a Makefile through my program in order to launch the tests generated by TestPr

Features

  • Scan a C project and detect automatically all functions
  • Generate a quick test for default type: int float bool and string
  • Generate a test that can test result made in the standard output made by a function
  • Generate a new makefile automatically to tests all functions
